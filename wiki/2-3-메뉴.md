# 간변 메뉴

---

## 3-1. 메인 메뉴

![image.png](media/image2.png)

1. **게임 시작**
    말 그대로 게임 시작 메뉴입니다.

2. **팀 선택**
    블루팀과 레드팀을 변경할 수 있는 메뉴입니다.

3. **맵 선택**
    맵을 선택할 수 있는 창을 표시합니다.

4. **서버 정보**
    서버에 제작자와 특정 맵에 제작자가 나와있습니다.

5. **무기고**
    ![image.png](./wiki/media/image3.png)

    상단에 버튼들은 각각 무기 유형의 무기고를 엽니다.

    1. 파츠 선택 메뉴
    2. 탄약 선택 메뉴
    3. 휴지통
    4. 근접무기 선택 메뉴
    5. 투척무기 선택 메뉴

6. **관리자 메뉴**
    관리자만 접근 가능한 메뉴를 엽니다. 관리자만 보입니다.

7. **스폰**
스폰으로 이동합니다.
