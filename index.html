Installation Guide
01
클라이언트 다운로드
이 페이지 하단의 Download 섹션에서 Windows 클라이언트 설치 파일(.exe)을 다운로드합니다.
Windows 전용
02
설치 파일 실행
다운로드된 .exe 파일을 실행합니다. 설치 마법사가 시작되면 화면의 안내에 따라 다음(Next)을 눌러 진행하세요.
화면 안내대로 진행
03
설치 완료 확인
설치가 완료되면 바탕화면 또는 시작 메뉴에 Min. At. Zero 아이콘이 생성됩니다.
04
정품 계정으로 로그인
클라이언트 실행 후 정품 Microsoft / Minecraft 계정으로 로그인합니다. 정품 계정 없이는 게임 시작이 불가합니다.
정품 계정 필수
05
이 페이지에서 게임 시작
설치 이후 게임 시작, 계정 관리, 리플레이 폴더 열기 등 모든 기능은 이 사이트의 런처 버튼을 통해 이용할 수 있습니다.

FAQ
네, 반드시 필요합니다. 게임 시작 시 정품 계정으로 로그인이 되어 있지 않으면 게임이 시작되지 않습니다.
클라이언트를 설치하면 마인크래프트 시작 화면이 바뀌며, 시작하기 버튼을 누르면 자동으로 서버에 접속됩니다. IP를 직접 입력하고 싶다면 아래 주소를 사용하세요.

tharu81.kro.kr
마인크래프트 자체는 Mac/Linux에서도 실행되지만, 전용 클라이언트가 현재 Windows만 지원합니다. Mac/Linux에서는 플레이가 불가합니다.
불가합니다. 서버 플레이에 필요한 모드팩은 전용 클라이언트를 통해서만 설치·다운로드할 수 있습니다. 또한 부정행위 방지를 위해 전용 클라이언트 외 접속은 허용되지 않습니다.
설치되는 모드팩에 다양한 최적화 모드가 이미 포함되어 있습니다. 그래도 성능이 낮다면 게임 내 그래픽 설정(렌더 거리, 그래픽 품질 등)을 낮춰 보시기를 권장합니다.
부정행위 방지를 위해 추가 모드 설치는 엄격히 금지되어 있습니다. 클라이언트는 설치된 모드에 대해 무결성 검사를 진행하므로 임의로 추가된 모드는 감지됩니다.
클라이언트 및 서버 관련 버그나 문제는 디스코드 @tharu_81 로 문의해 주세요.
정기 점검은 따로 없습니다. 다만 치명적인 오류 발생 시 예고 없이 점검이 진행될 수 있습니다. 점검 공지는 디스코드에서 확인하세요.
