# Min. At. Zero Web Client

본 프로젝트는 기존 데스크톱 프로그램에서 처리하던 기능을  
웹 클라이언트로 이전하여,

- 게임 실행
- 로그인 및 계정 관리
- 마인크래프트 초기화 및 관리

등의 기능을 제공하는 것을 목표로 합니다.

본 저장소는 **배포용 웹 클라이언트에 대한 안내 목적**으로 공개되어 있으며,  
실제 서비스에 사용되는 **원본 소스 코드는 비공개로 관리**됩니다.

본 프로젝트와 관련된 소스 코드, 리소스 및 결과물에 대한  
무단 수정, 재배포, 2차 가공은 허용되지 않습니다.
